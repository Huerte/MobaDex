About Mobadex

Mobadex is a Mobile Legends hero matchup analyzer that provides real-time insights into hero counters, synergies, and meta trends. It helps players draft more strategically for better in-game performance.

Mobadex was created for the Ghost Empire Nation (GEN.) squad to support team members in improving their performance. By offering tactical drafting insights and real-time data, it aims to give every match a strategic edge.

  • Real-time matchup analytics
  • Hero counters & synergy logic
  • Optimized for mobile devices
  • Meta-aligned suggestions

Developed using Django (Python) and Bootstrap 5, Mobadex leverages a modern, API-driven architecture with an emphasis on responsiveness, performance, and accessibility.

Crafted quietly in service of better team coordination.
